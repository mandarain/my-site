# Hello, World

这是个人网站的第一篇文章。

## 为什么要建站

在这之前，我的作品散落在不同平台——Twitter、Behance、GitHub……没有一个真正属于自己的空间。我想找一个地方，可以按自己的方式来组织内容，不需要被算法推荐绑架。

## 技术选型

选 Next.js + Tailwind CSS 的理由很简单：

1. 静态生成，速度快
2. 可以写 Markdown，维护成本低
3. 从 Vercel 部署几乎零配置

## 接下来

- 项目复盘与技术笔记
- 音乐与创作相关随笔
- 一些不太成熟的胡思乱想

希望这里能慢慢长成一个有趣的角落。
