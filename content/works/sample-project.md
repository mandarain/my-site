# 探索即兴钢琴

这个项目探索了即兴钢琴与算法生成之间的边界。

## 灵感

即兴演奏的本质是即时决策——在已知与未知之间找到一条旋律线。我想通过代码来模拟这种过程。

## 实现

使用 MIDI 协议 + 实时 Web Audio API：

- 随机行走生成音高序列
- 马尔可夫链控制节奏模式
- 用户可通过滑块调节“随机性”参数

## 后续方向

- 加入更复杂的和声约束
- 支持录音回放与导出
- 多人即兴协作模式
